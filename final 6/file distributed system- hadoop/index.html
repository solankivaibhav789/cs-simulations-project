FILE DISTRIBUTION SYSTEM

In Hadoop distribution system, this work on two node

1)Name Node

2)Data Node

1)Name Node:- name node work is divide file into 64 mb or multiple of 64mb block

than it transfer each block into different server such that same node are not repeated on same server. and get back to address of the data node where each block is available.

2)Data Node:- data node work for store the data.

if any process come for do operation on file than name node divide process code and send code to data node.

than data node do process individually and after completing the process combine all output into one and send back to the name node.
